# DRPC Integration

DRPC (Deesse RPC) is how you add custom API procedures to your Deesse app. Built on `@deessejs/server`, it provides type-safe queries, mutations, and a Next.js adapter via `@deessejs/server-next`.

## Packages

| Package | Version | Purpose |
|---------|---------|---------|
| `@deessejs/server` | 1.2.0 | Core RPC protocol |
| `@deessejs/server-next` | 1.3.1 | Next.js adapter |
| `@deessejs/fp` | * | Result type (`ok`, `err`) |
| `zod` | ^4.0.0 | Validation (peer dependency) |

## Workflow

```
1. Define procedures   →  src/server/index.ts
2. Register in config  →  deesse.config.ts (routes option)
3. Route handler       →  auto-generated by @deessejs/next
4. Use from client     →  React hooks or fetch
```

## Step 1: Define Procedures

```typescript
// src/server/index.ts
import { defineContext, createAPI, createPublicAPI } from "@deessejs/server";
import { deesse } from "@/lib/deesse";
import { z } from "zod";
import { ok, err } from "@deessejs/fp";

const { t, createAPI } = defineContext({
  context: {
    db: deesse.database,
    auth: deesse.auth,
  },
});

const appRouter = t.router({
  users: {
    list: t.query({
      args: z.object({ limit: z.number().default(10) }),
      handler: async (ctx, args) => {
        const users = await ctx.db.query.users.findMany({ limit: args.limit });
        return ok(users);
      },
    }),
    create: t.mutation({
      args: z.object({ name: z.string(), email: z.string().email() }),
      handler: async (ctx, args) => {
        const user = await ctx.db.insert(schema.users).values(args);
        return ok(user);
      },
    }),
  },
});

export const api = createAPI({ router: appRouter });
export const publicAPI = createPublicAPI(api);
export type AppRouter = typeof appRouter;
```

## Step 2: Register in Config

Pass `publicAPI` to `defineConfig`:

```typescript
// deesse.config.ts
import { defineConfig } from "deesse";
import { publicAPI } from "@/server";

export const config = defineConfig({
  database: drizzle({ connection: process.env.DATABASE_URL! }),
  secret: process.env.DEESSE_SECRET!,
  auth: { baseURL: "http://localhost:3000" },
  routes: publicAPI,
});
```

The route handler is auto-generated by `@deessejs/next` and merges all routes into a single handler. See [README.md](./README.md) for the full architecture.

## Step 3: Use from Client

### React Hooks

```typescript
// app/users/page.tsx
"use client";
import { createHooks } from "@deessejs/server-react";
import { publicAPI } from "@/server";

const { useQuery, useMutation } = createHooks(publicAPI);

export default function UsersPage() {
  const { data, isLoading } = useQuery("users.list", { limit: 10 });

  const createUser = useMutation("users.create");

  if (isLoading) return <div>Loading...</div>;

  return (
    <div>
      {data?.map((user) => (
        <div key={user.id}>{user.name}</div>
      ))}
      <button onClick={() => createUser({ name: "John", email: "john@example.com" })}>
        Add User
      </button>
    </div>
  );
}
```

### Direct Fetch

```typescript
// Call from anywhere
const result = await fetch("/api/users.list", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ args: { limit: 10 } }),
}).then((r) => r.json());

// { ok: true, value: [...] }
// or
// { ok: false, error: { code: "NOT_FOUND", message: "..." } }
```

## Server vs Client API

| Export | Where | Internal Ops | HTTP Exposed |
|--------|-------|--------------|--------------|
| `api` | Server only | Yes | No |
| `publicAPI` | Client + Server | No | Yes |

```typescript
// Server component — full access
import { api } from "@/server";

await api.users.list({ limit: 10 });        // query OK
await api.users.delete({ id: "123" });      // internalMutation OK (server only)

// Client component — public API only
import { publicAPI } from "@/server";

await publicAPI.users.list({ limit: 10 }); // query OK
await publicAPI.users.delete({ id: "123" }); // TypeScript error! (internal)
```

## Procedure Types

| Type | HTTP Exposed | Server Only | Use Case |
|------|-------------|-------------|----------|
| `t.query()` | Yes | Yes | Public reads |
| `t.mutation()` | Yes | Yes | Public writes |
| `t.internalQuery()` | **No** | Yes | Admin reads |
| `t.internalMutation()` | **No** | Yes | Admin writes |

## Shared Types

Export the router type for end-to-end type safety:

```typescript
// src/server/index.ts
export type AppRouter = typeof appRouter;

// Then import in client:
import type { AppRouter } from "@/server";
```

## Quick Reference

```typescript
// Define
t.query({ args, handler })              // Public read
t.mutation({ args, handler })            // Public write
t.internalQuery({ handler })           // Server-only read
t.internalMutation({ args, handler })   // Server-only write

// Create
api = createAPI({ router: appRouter })      // Full API (internal ops)
publicAPI = createPublicAPI(api)            // Client-safe (public only)

// Register
defineConfig({ ..., routes: publicAPI })    // In deesse.config.ts

// Consume
fetch("/api/users/list", {                  // Direct HTTP
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ args: { limit: 10 } }),
})
```
