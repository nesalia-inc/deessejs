# Getting Started

Welcome to DeesseJS! This guide will help you get started with building modern CMS applications using DeesseJS and Next.js.

## What You'll Learn

DeesseJS is a headless CMS framework built specifically for Next.js that provides collection-based content management, visual page editing, and auto-generated APIs - all with full TypeScript support.

### Explore the Documentation

- **[What is DeesseJS?](/docs/what-is-deessejs)** - Learn about the framework and its features
- **[Comparison](/docs/comparison)** - See how DeesseJS compares to other CMS solutions

## Quick Start

Get up and running with DeesseJS in just a few minutes:

### 1. Create a New Project

```bash
npx create-deessejs-app my-app
cd my-app
```

### 2. Start the Development Server

```bash
npm run dev
```

### 3. Open Your Browser

Navigate to [http://localhost:3000](http://localhost:3000) to see your application.

## Prerequisites

Before you begin, make sure you have:

- **Node.js** 18+ installed
- Basic knowledge of **Next.js** and **React**
- Familiarity with **TypeScript**
